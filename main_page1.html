About

This is a maze-solving fun game devoloped by us to enjoy in the free time. This game is devoloped on the theme of finding the shortest path between 2 points in a region to make sure the person playing this has a curiousity of finding new ways. This shortest path is found with the help of A-star Algorithm.

Rules & Regulations

1. Each person gets to play only once.
2. Each person's main aim is to get the virtual character to first solve the maze.
3. Secondly, the other aim is to solve the maze with the shortest path as possible.
4. The person's aim is not to obtain the highest points, but to reach the minimum points i.e. equals to zero.
5. The person doesnot have any timer going on at the moment but less the time, more the better.
6. This game is made for every part of society. Everyone will be able to play the game.
